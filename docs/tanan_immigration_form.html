Complete Field Analysis — Source Document Mapping
Field Name Source Document(s) Notes
PERSONAL DETAILS — Page 1
Family Name / SurnamePASSPORT NAT. ID BIRTH CERTPassport is primary — most reliable. Cross-validate with National ID and Birth Certificate for consistency.
Given NamesPASSPORT NAT. ID BIRTH CERTExtract exactly as printed — do not reformat or translate.
Telephone NumberMANUAL ONLYNo official document contains the applicant's personal phone number. Must be entered by applicant.
Email AddressMANUAL ONLYNo official document contains personal email. Must be entered by applicant.
Passport NumberPASSPORTUnique to passport. High confidence extraction from MRZ line 1.
Country of IssuancePASSPORTFound on passport data page and MRZ.
Passport Issue DatePASSPORTClearly printed on passport data page.
Passport Expiry DatePASSPORTCritical field — high confidence threshold required. Also in MRZ line 2.
Country of ResidencePROOF OF ADDRESS MANUALProof of address provides current address/country. However applicant may currently reside abroad — verify manually.
Place of BirthPASSPORT BIRTH CERTBirth certificate is more reliable. Passport sometimes abbreviates city names.
Marital StatusMANUAL ONLYNo standard document reliably states marital status. Some national IDs include it but inconsistently. Mark manual.
Date of MarriageMANUAL ONLYWould require marriage certificate — not in the standard document list. Manual entry.
Date of BirthPASSPORT NAT. ID BIRTH CERTCross-validate across all three documents. Discrepancies are a red flag for the agent.
CitizenshipPASSPORT NAT. IDPassport nationality field is the most reliable source.
Eye ColorMANUAL ONLYAppears on some national IDs and driving licences but not consistently. Mark manual — applicant fills.
HeightMANUAL ONLYAppears on some national IDs and driving licences but not in the standard document list. Manual entry.
Current OccupationEMPLOYMENT LETTER MANUALEmployment letter provides job title and employer. If unemployed or self-employed, must be entered manually.
Current Status in CanadaMANUAL ONLYVisitor / Student / Worker / PR / Citizen — no document in the standard list contains this. Manual entry.
Native LanguageMANUAL ONLYNo standard document contains native language. Manual entry by applicant.
Current Status ExpiryMANUAL ONLYWould require the applicant's Canadian visa or permit document — not in the standard document list.
Referred ByMANUAL ONLYInternal referral tracking — no document source. Manual entry.
Number of ChildrenMANUAL ONLYNo standard document reliably states number of children. Manual entry.
Course Start DateMANUAL ONLYWould require a study permit or offer letter — not in the standard document list. Manual entry.
Course End DateMANUAL ONLYSame as above. Manual entry.
Initially Entered Canada asMANUAL ONLYEntry category (Visitor/Refugee/Student/Worker) — requires travel history knowledge. Manual entry.
UCI NumberMANUAL ONLYUnique Client Identifier from IRCC. Not printed on passport. Applicant must know this from previous IRCC correspondence.
Date First Entered CanadaMANUAL ONLYPassport may show entry stamps but extraction of specific stamps is unreliable. Manual entry.
Port of EntryMANUAL ONLYSame as above. Manual entry from memory or travel documents.
Deported / Refused Visa / EntryMANUAL ONLYYes/No declaration. Personal knowledge only. Manual entry.
Applied to IRCC beforeMANUAL ONLYYes/No declaration. No document source. Manual entry.
Applied to any PNP beforeMANUAL ONLYYes/No declaration. No document source. Manual entry.
Relative in CanadaMANUAL ONLYYes/No declaration. Personal knowledge only. Manual entry.
Highest Education CompletedMANUAL ONLYCanadian equivalency requires applicant self-assessment. Transcripts/diplomas could inform but are not in the standard document list.
Total Years of EducationMANUAL ONLYCalculated from educational history. Manual entry.

Summary — What AI Can and Cannot Fill
✓ Fields AI can pre-fill from documents
Surname, Given Names, Date of Birth, Place of Birth, Citizenship, Nationality
Passport Number, Issue Date, Expiry Date, Country of Issuance, MRZ Lines
Current Address, City, State, Postal Code (from Proof of Address)
Employer Name, Job Title, Employment Start Date (from Employment Letter)
Employer Address (from Employment Letter letterhead)
National ID Number (from National ID card)
✗ Fields that must always be entered manually
Phone Number, Email Address, Eye Color, Height
Marital Status, Date of Marriage, Native Language
Number of Children, UCI Number, Current Status in Canada
All Yes/No declarations (IRCC, PNP, deportation, relatives)
Entire Education History (no education docs in required list)
English Test Scores (no test report in required list)
All past employment beyond current job
All past addresses beyond current address
All family member details (spouse, children, siblings, parents)
Course dates, referral source, Canada entry dates
Recommendation for Tanan Immigration: Consider adding the following documents to the required upload list to increase auto-fill coverage significantly: Educational Certificates / Transcripts (fills education history), IELTS / CELPIP Score Report (fills language test section), Spouse's Passport (fills spouse columns in family tables), Marriage Certificate (fills marital status and date of marriage). Adding these 4 document types would increase AI-fillable fields from approximately 25% to approximately 55% of the total form.
